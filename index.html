🎼 ORCHESTRA

손으로 연주하는 인터랙티브 오케스트라 — 사용 가이드

시작하는 법

1. 카메라 시작
화면 중앙의 카메라 시작 버튼을 클릭합니다. 웹캠 권한 허용 후 악기 샘플이 로딩됩니다 (인터넷 연결 필요, 약 5~10초).
2. 손 올리기
웹캠 앞에 손을 올리면 자동으로 인식됩니다. 밝은 배경, 충분한 조명에서 잘 작동합니다.

손 조작법

🎻 왼손 좌우
화면 하단 8가지 화음 중 하나를 선택합니다. 현악 앙상블 소리가 납니다.
🎵 오른손 좌우
멜로디 음을 선택합니다 (A단음계 12음). 플루트 소리가 납니다.
양손 위아래
손을 위로 올릴수록 소리가 커지고, 내릴수록 작아집니다.
손 빼기
화면 밖으로 손을 치우면 해당 소리가 서서히 멈춥니다.

연주 팁

🎵 초보자 추천 진행

왼손을 Am → F → C → G 순서로 천천히 이동하면서 오른손으로 멜로디를 얹어보세요. 수천 곡에 쓰이는 가장 아름다운 진행입니다.

🎻 풍성한 소리 내기

왼손을 한 화음에 고정한 채 손을 위로 높이 올려 볼륨을 키우고, 오른손으로 천천히 멜로디를 그려보세요. 둘 다 화면 중간 높이에 두면 가장 균형 잡힌 소리가 납니다.

💡 잘 안 될 때

조명이 밝을수록 손 인식이 잘 됩니다. 손바닥이 카메라 쪽을 향하게 하고, 배경이 단순할수록 정확도가 높아집니다.
